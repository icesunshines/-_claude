# 项目说明书

> 基于 FastAPI + Vue 3 的医疗健康数据可视化与风险预测系统

---

## 一、项目概述

本项目利用天池精准医疗大赛的两份医疗数据集，构建血糖预测（回归）和妊娠期糖尿病预测（分类）模型，通过 FastAPI 提供 REST API，Vue 3 + ECharts 构建可视化大屏，支持深色/浅色主题切换、六维度健康评估、实时数据刷新和角色权限管理。

### 1.1 核心功能

| 功能 | 说明 |
|------|------|
| 登录注册 | JWT 认证、角色权限（管理员/普通用户） |
| 可视化大屏 | 深色/浅色主题、六边形雷达图、血糖趋势、SHAP特征重要性、风险仪表盘、风险等级卡片 |
| 风险预测 | 输入体检指标，预测血糖值并评估风险等级 |
| 糖尿病分类 | 输入基因+临床指标，预测妊娠期糖尿病风险 |
| 智能问答 | 医患对话助手（预留） |
| 个人中心 | 查看历史预测记录 |
| 管理面板 | 管理员专用，查看系统统计 |

### 1.2 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 后端 | FastAPI + Uvicorn | 0.111.0 / 0.30.1 |
| 前端 | Vue 3 + Vite + Element Plus | 3.4 / 5.4 / latest |
| 图表 | ECharts 5 | 5.5 |
| ML | LightGBM + scikit-learn | 4.6.0 / 1.5.0 |
| 数据 | pandas + numpy | 2.2.2 / 1.26.4 |

### 3.1 数据预处理模块 (`src/preprocessing.py`)

负责将原始医疗数据转换为模型可用的格式。

#### 初赛数据预处理 (`preprocess_initial`)

```python
def preprocess_initial(df: pd.DataFrame) -> pd.DataFrame:
```

**处理流程：**

1. **丢弃高缺失列**：乙肝五项（缺失率75.8%）
2. **性别编码**：男=0, 女=1
3. **删除非特征列**：`id`, `体检日期`
4. **缺失值填充**：中位数填充
5. **标准化**：StandardScaler（排除标签列"血糖"）

**输出**：处理后的 DataFrame，形状约 (5642, 35)，含"血糖"列

#### 复赛数据预处理 (`preprocess_final`)

```python
def preprocess_final(df: pd.DataFrame) -> tuple[pd.DataFrame, pd.Series]:
```

**处理流程：**

1. 删除缺失率 > 80% 的列
2. 保留 SNP 列，删除缺失率 > 50% 的 SNP
3. 数值转换：将 object 类型转为数值
4. 缺失值填充：中位数填充
5. 分离特征和标签：X 和 y
6. 标准化：StandardScaler

**输出**：(X, y)，X 形状 (1000, 76)，y 形状 (1000,)

api.interceptors.response.use(

      localStorage.removeItem('user')
      window.location.href = '/login'
    }
    return Promise.reject(error)
  }
)

### 6.2 前端导入路径错误

- `views/` 下的组件：`import ... from '../api/request'`

### 6.3 前端跨域错误

确保 `vite.config.js` 中配置了 `/api` 代理

### 6.4 后端启动失败

## 八、毕设答辩要点

### 8.1 核心亮点

1. **双模型架构**：同时实现回归（血糖预测）和分类（糖尿病预测）
2. **医疗数据预处理**：处理高缺失率、多类型医疗指标
3. **沉浸式可视化大屏**：6 个 ECharts 图表 + 深色/浅色主题 + 动态效果
4. **前后端分离**：FastAPI + Vue 3 + Element Plus 现代架构
5. **完整工程化**：JWT 认证、角色权限、响应式布局、错误处理
6. **完整流程**：从数据清洗 → 模型训练 → API 服务 → 前端展示

### 8.2 数据说明

- **初赛数据**：5642条体检记录，39项指标，预测连续血糖值
- **复赛数据**：1000条孕妇记录，55个SNP基因位点，预测二分类

### 8.3 模型性能

| 任务 | 指标 | 结果 |
|------|------|------|
| 血糖预测 | RMSE | 1.1445 |
| 糖尿病分类 | AUC | 0.8368 |

### 8.4 技术难点

1. **中文路径兼容**：LightGBM 模型保存路径问题
2. **高缺失数据处理**：乙肝五项75%缺失，SNP位点部分高缺失
3. **前后端字段映射**：前端友好字段名 → 模型特征名
4. **主题系统**：CSS 变量实现深色/浅色切换，避免全局样式污染
5. **ECharts 自适应**：所有图表监听窗口 resize 自动重绘
6. **图标兼容**：Element Plus 图标版本差异处理
